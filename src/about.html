Dropout Prevention Data

Save the Children work toward securing education for all children.
A challenge is that many children drop out of school around the world,
especially in african countries.Save the Children receives a lot of data,
and our mission is to help Save the Children analyze and understand
this data, so they can use the data to prevent children from dropping out
of school.
Our solution to this problem is a platform we call HDX
(Humanitarian Data Exchange). The Humanitarian Data Exchange (HDX)
is an open platform for sharing humanitarian data. We want to add
geographical data to the attendance and enrollment data in developing
countries. Combining this new data set with other geographical
humanitarian data shared on the HDX (Humanitarian Data Exchange)
will enable Save the Children to detect and understand underlying social
and economic problems contributing to children dropping out. An example
of this would be to cross-reference data on food price and availability,
with attendance data for students from a given region.
This would make it possible to intervene earlier to ensure continued
enrollment and attendance. The use of HDX-data will be facilitated by
the HDX Python API, which makes it possible to both use and add data to the HDX-database.

About us

This project originated from an innovation camp held at OsloMet university. We are a computer science student group that was posed with the question on how to help Save the children with the challenges they face. One of these challenges is that Save the Children gets a lot of data, but they don't know how to use it efficiently. We tried to come up with innovative solutions to this question, and on this page you will find information about our solution.
